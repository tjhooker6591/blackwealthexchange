# BWE Business + Seller GTM

## Business GTM
- onboarding wizard: profile completion + trust score
- share-your-profile toolkit (badge, QR, social cards)
- review capture cadence (post-visit/purchase)
- sponsored listing upsell

## Seller GTM
- onboarding sequence: account -> first product -> first order
- category discoverability + curated collections
- creator integration for product amplification
- repeat buyer prompts and review collection

## Funnel stages
1. Acquire (outreach + partner referrals)
2. Activate (first meaningful setup)
3. Convert (first lead/order)
4. Retain (repeat activity)
5. Amplify (share/referral loops)

## Messaging pillars
- revenue-focused value
- trusted discovery
- culturally aligned demand
- measurable outcomes

## KPI set
- business activation rate
- seller activation rate
- time to first product
- time to first order
- profile shares/reviews
- repeat buyer rate
