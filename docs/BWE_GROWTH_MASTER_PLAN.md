# BWE Growth Master Plan

## Target context

- Long-range framing: 0.1%–0.5% of projected Black buying power.
- Operating principle: compounding network effects, not vanity traffic.

## Positioning

Discover, buy, hire, learn, and grow inside one trusted Black economic ecosystem.

## Stakeholder acquisition model

- Users/buyers: search utility + trust + local discovery + referrals
- Businesses: profile-as-acquisition-node + reviews + sponsored uplift
- Sellers: storefront activation + conversion + repeat buyer loops
- Employers: hiring solution + intake + pipeline
- Employees/talent: network visibility + applications + opportunities
- Creators/affiliates: campaign distribution + attributable outcomes

## Core loops

1. User referral loop
2. Creator distribution loop
3. Business-to-customer loop
4. Seller-to-buyer loop
5. Employer-to-talent loop

## Channel strategy

- Creator & ambassador program (HBCU + city + niche creators)
- Local SEO (city/category/near-me pages + reviews)
- Content authority (guides, founder profiles, collections)
- Partnerships (HBCUs, chambers, media, community orgs)

## Wedge strategy

Choose first wedge from live platform evidence:

- city/category density
- supply readiness
- repeat demand potential
- creator/partner leverage
- 30–60 day transaction probability

## KPI model

Track weekly:

- visitor->signup
- signup->first-action
- signup->first-purchase
- referral share
- business/seller/employer activation
- GMV by segment
- AOV + repeat purchase
- creator-attributed GMV
- employer repeat post rate

## Build-now vs build-later

- Build-now: referral primitives, creator attribution, share/review mechanics, wedge campaigns
- Build-later: expanded geos/categories, advanced recommendation and social mechanics
